# 기본 문법 배우기
## 2-1 연산자 우선 순위
- 20 -> 가장 높음 1 -> 가장 나중에
- 20 : `() 그룹화`
- 19 : `., [], new, () 함수호출`
- 18 : `new(인수 없이)`
- 17 : `++(후위), --(후위)`
- 16 : `!, ~, +(단항), -(단항), ++(전위), --(전위), typeof, void, delete, await`
- 15 : `**`
- 14 : `*, /, %`
- 13 : `+(다항), -(다항)`
- 12 : `<<, >>, >>>`
- 11 : `<, <=, >, >=, in, instanceof`
- 10 : `==, !=, ===, !==`
- 09 : `&`
- 08 : `^`
- 07 : `|`
- 06 : `&&`
- 05 : `||`
- 04 : `? :` (삼항 연산자)
- 03 : `=, +=, -=, **=, *=, %=, /=, <<=, >>=, >>>=, &=, ^=, !=`
- 02 : `yield`, `yield*`
- 01 : `,`(쉼표)
